About

This demo showcases a simple Flask app shipped via GitHub Actions to AWS (ALB/ASG) and observed with Prometheus & Grafana. It’s intentionally minimal so you can focus on pipeline behavior instead of app complexity.

Stack

Flask · Gunicorn · Nginx/ALB · ASG · GitHub Actions · Prometheus · Grafana

Goals

Fast, reliable deployments; easy rollbacks; transparent visibility of service health.
